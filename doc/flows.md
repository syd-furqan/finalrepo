# Faculty / Student Features (Afnan)

## US-06: Guest Access Request Portal
Users can submit guest access requests through a form.

## US-07: Request Approval Notifications
Users receive notifications when their request is approved or rejected.

## US-08: Vehicle Registration Request
Staff can register vehicles.

## US-09: View Access Request Status
Users can track request status.

## US-10: Generate Guest Pass
User generates a QR guest pass.

## US-11: Set Expiry
User sets expiry time for pass.

## US-12: Cancel Guest Pass
User can cancel a pass.
