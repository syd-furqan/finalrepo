# Campus Gate Access System — Product Backlog

**Team GLITCH** | CS360 Software Engineering  
**Requirement Lead:** Muhammad Aaffan Khan Niazi

---

| ID | User Story | Story Points | Risk Level | Release |
|----|-----------|:------------:|:----------:|---------|
| US-01 | As a **Security Guard**, I want to view a real-time dashboard of all active entry requests so that I can manage gate traffic efficiently. | 5 | Medium | Checkpoint |
| US-02 | As a **Security Guard**, I want to search and look up visitor information by name, ID, or vehicle number so that I can quickly verify who is requesting entry. | 3 | Low | Checkpoint |
| US-03 | As a **Security Guard**, I want to perform real-time credential verification against the university database so that only authorized individuals are granted access. | 8 | **High** | Checkpoint |
| US-04 | As a **Security Guard**, I want to log every entry and exit event with a timestamp so that a complete access record is maintained at all times. | 5 | Medium | Checkpoint |
| US-05 | As a **Security Guard**, I want to flag or deny an entry request and add a reason so that security incidents are documented immediately. | 3 | Low | Final |
| US-06 | As a **Faculty** member, I want to submit an access request for a guest or visitor through an online portal so that I can sponsor their entry without visiting the gate in person. | 5 | Low | Checkpoint |
| US-07 | As a **Faculty** member, I want to receive a notification when my access request is approved or denied so that I can inform my visitor promptly. | 2 | Low | Final |
| US-08 | As a **Staff** member, I want to submit a vehicle registration request so that my vehicle is pre-approved for campus entry. | 3 | Low | Final |
| US-09 | As a **Staff** member, I want to view the status of my pending and past access requests so that I know whether my submissions have been processed. | 2 | Low | Final |
| US-10 | As a **Student**, I want to generate a single-use guest pass for a visitor so that my guest can enter campus without requiring faculty sponsorship. | 5 | Medium | Final |
| US-11 | As a **Student**, I want to set an expiry time on a guest pass so that the pass is automatically invalidated after the visit. | 3 | Medium | Final |
| US-12 | As a **Student**, I want to cancel a guest pass that I previously issued so that I can revoke access if plans change. | 2 | Low | Final |
| US-13 | As a **Security Admin**, I want to view a full security audit log of all gate activity so that I can investigate incidents and ensure compliance. | 8 | **High** | Checkpoint |
| US-14 | As a **Security Admin**, I want to add, edit, or deactivate user accounts (guards, faculty, staff, students) so that access permissions stay up to date. | 5 | Medium | Final |
| US-15 | As a **Security Admin**, I want to export audit logs as a CSV or PDF report so that I can share findings with university management. | 3 | Low | Final |
| US-16 | As a **Security Admin**, I want to configure credential verification rules (e.g., ID expiry, banned lists) so that the system enforces the latest security policies. | 8 | **High** | Final |
| US-17 | As a **Security Guard**, I want to scan a QR-code-based guest pass so that visitor verification is fast and error-free. | 5 | Medium | Final |
| US-18 | As a **Security Admin**, I want to receive automated alerts for repeated failed credential checks so that potential intrusion attempts are flagged in real time. | 8 | **High** | Final |
